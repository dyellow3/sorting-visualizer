## Local Testing

1. Clone the repository:

   ```bash
   git clone https://github.com/dyellow3/sorting-visualizer.git
2. Navigate to the project directory
    ```bash
   cd sorting-visualizer
3. Install dependencies
   ```bash
   npm install
4. Start the development server
   ```bash
   npm run dev
